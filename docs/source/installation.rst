Installation
============

Prerequisites
-------------

- Python 3.12 or higher
- Pygame (will be installed automatically)

Using pip
---------

.. code-block:: bash

   pip install logngraph

From source
-----------

.. code-block:: bash

   git clone https://github.com/GS11566/logngraph.git
   cd logngraph
   pip install -e .

Verifying installation
----------------------

.. code-block:: python

   import logngraph
   print(logngraph.__version__)
